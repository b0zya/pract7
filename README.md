# Lunar Station — посадка модуля

Небольшая аркадная игра на Pygame: нужно посадить модуль на зеленую
посадочную платформу, аккуратно управляя двигателем и расходуя топливо.

## Возможности

- Свободное падение корабля под действием гравитации.
- Управление двигателем стрелкой **Вверх** — гасит скорость падения,
  расходуя топливо (1 единица за кадр).
- Отображение текущего запаса топлива и скорости падения на экране.
- Посадочная платформа (зеленая линия).
- Проверка условий посадки: скорость меньше 3 px/кадр и попадание
  в зону платформы — успех, иначе — крушение.
- Если топливо заканчивается в полете — корабль падает неуправляемо.
- Кнопка «Перезапуск» для повторной попытки без перезапуска программы.

## Архитектура проекта

- `lander.py` — класс `Lander`: положение, скорость, топливо, отрисовка
  корабля. Ничего не знает про платформу или интерфейс — только физика.
- `game.py` — класс `Game`: окно, HUD (топливо/скорость), кнопка
  перезапуска, посадочная платформа, проверка условий посадки,
  всплывающие сообщения о результате и основной игровой цикл.

## Установка зависимостей

```
python -m venv .venv
source .venv/bin/activate      # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

> Используется `pygame-ce` (Community Edition) вместо классического
> `pygame`, так как он поддерживает более новые версии Python (3.10–3.14).
> Модуль импортируется так же: `import pygame`.

## Запуск

```
python game.py
```

## Управление

- **Стрелка Вверх** — включить двигатель (тормозит падение, тратит топливо)
- **Клик мышью по кнопке «Перезапуск»** — начать уровень заново

## Условия посадки

- Скорость касания платформы **меньше 3 пикселей за кадр** — «Успешная посадка!»
- Скорость **3 пикселя за кадр и больше**, либо касание за пределами платформы — «Корабль разбился!»
